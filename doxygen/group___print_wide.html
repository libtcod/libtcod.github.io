Detailed Description

Deprecated printing functions which use UCS-2 or UTF-32 wchar_t characters.

Despite the _utf postfix these functions do not handle variable length characters as you'd expect from UTF-16. They only support full Unicode when the platform has a 4 byte wchar_t. Printing (Unicode) should be used instead.
